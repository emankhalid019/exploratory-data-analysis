 📊 EDA with Python

Welcome to my **EDA with Python** repository!

This repository documents my journey of learning **Exploratory Data Analysis (EDA)** using Python. It contains practice notebooks, datasets, and examples that help me understand how to explore, clean, analyze, and visualize data effectively.



📖 About

Exploratory Data Analysis (EDA) is an important step in data science that helps understand the structure, patterns, relationships, and quality of data before building machine learning models.

This repository includes my learning, hands-on exercises, and mini projects completed while practicing EDA with Python.
